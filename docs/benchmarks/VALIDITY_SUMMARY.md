# Validity Summary: Three-Level Difficulty Framework

## 論理的根拠の概要

### 1. 三段階選択の根拠

```
二分法 ──→ 情報損失（Medium領域の欠落）
四分法 ──→ 過剰分割（サンプル数不足）
三段階 ──→ Goldilocks Zone（最適バランス）
```

**根拠**: Bloomの分類学、認知負荷理論、教育評価の先行研究

### 2. 段階定義の理論的基盤

| 段階 | 理論的基盤 | 測定対象 |
|------|-----------|---------|
| Easy | 語彙的マッチング | 索引の直接参照能力 |
| Medium | ブール論理 + ランキング | 複合クエリ処理能力 |
| Hard | 意味的・文脈的推論 | 高度情報ニーズ対応能力 |

### 3. 実証的裏付け

v2.0評価結果が理論的予測と一致：
- Easy高性能（90%）→ 基盤機能は実装済
- Medium中性能（50%）→ 部分的に実装
- Hard低性能（40%）→ 未実装機能の影響

### 4. 妥当性の3側面

- **内部妥当性**: 同層内で一貫した挙動
- **外部妥当性**: 実開発者検索ログと相関
- **構造妥当性**: 情報検索理論と一致

## 結論

三段階分けは統計的・認知的・実用的・理論的に妥当である。
